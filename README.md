# Elegant Context Shopping App

A modern React application built with Vite that demonstrates shopping cart functionality using React Context API.

## Features

- Product catalog with images and pricing
- Interactive shopping cart with modal display
- Context-based state management for cart operations
- Responsive design with clean UI

## Tech Stack

- **React 19** - Frontend framework
- **Vite** - Build tool and development server
- **Context API** - State management
- **ESLint** - Code linting

## Installation

1. Clone the repository:

   ```bash
   git clone <repository-url>
   cd elegant-context-shopping-app
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`

## Available Scripts

- `npm run dev` - Start the development server
- `npm run build` - Build the project for production
- `npm run lint` - Run ESLint for code quality checks
- `npm run preview` - Preview the production build locally

## Project Structure

```
src/
├── components/     # Reusable UI components
├── store/         # Context providers for state management
├── assets/        # Static assets (images)
├── App.jsx        # Main application component
└── main.jsx       # Application entry point
```

## Usage

- Browse products in the shop
- Add items to cart by clicking "Add to Cart"
- View cart contents by clicking the cart icon in the header
- Remove items or adjust quantities in the cart modal
